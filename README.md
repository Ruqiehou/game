# CK-Style Grand Strategy Engine

Python 实现的十字军之王风格大战略模拟引擎。  
默认场景：**1066 英格兰**（诺曼征服前夜）。

作者：Ruqiehou  
更新：2026-07-15

---

## 运行环境

- Python 3.10+
- 标准库即可（无第三方依赖）

---

## 快速开始

### 可操作地图 UI（推荐）

```bash
python -m ck_engine.ui.server
```

浏览器打开：http://127.0.0.1:8765/

- 点击省份查看详情  
- 选中己方军团后再点省份 = 行军  
- 侧栏：征召 / 宣战 / 宴会 / 存读档 / 推进时间  
- 推进时间时 **跳过玩家 AI**，由你手动操作  

### 控制台自动模拟

```bash
python run_game.py
```

或：

```bash
python -m ck_engine.game.main
```

控制台加载 1066 场景，自动模拟约 2 年，输出统治者、战争、日志等。

```text
game/
├── run_game.py
├── README.md
├── saves/                   # 存档目录（运行后生成）
└── ck_engine/
    ├── core/                # 日期、属性、特质
    ├── world/               # 人物、地图、头衔、王朝、经济/继承
    ├── politics/            # 外交、内阁、派系、法律、阴谋
    ├── military/            # 军团、战斗、围城、战争

    ├── ai/                  # AI 性格与月度决策
    ├── game/                # 场景、模拟主循环、CLI
    └── ui/
        ├── server.py        # HTTP 服务 :8765
        ├── api.py           # 状态与玩家指令
        ├── map_layout.py    # 省份坐标
        └── static/          # 前端页面
```

## 已实现系统

| 模块 | 内容 |
|------|------|
| 世界 | 人物、王朝、头衔、邻接地图、税收/征召、生育、死亡与继承 |
| 法律 | 继承法 `pick_heir`（长子/幼子/长老等）、王权税加成 |
| 政治 | 外交（同盟禁战/参战、停战、宣称）、内阁、派系、阴谋 |
| 军事 | 征召、行军、战斗、围城、占城同步头衔、战争分数与和约 |
| 事件 | 月度触发与 AI 自动决议 |
| AI | 宣战、结盟、婚姻、宴会、发展、阴谋；**跳过玩家** |
| 场景 | 14 省、英格兰/诺曼底主要贵族 |
| UI | SVG 地图、选省/军团、时间推进、存档读档 |
| 存档 | `saves/autosave.json` 轻量快照 |

## 地图操作说明

| 操作 | 方式 |
|------|------|
| 选省份 | 点击地图多边形 |
| 选军团 | 点击地图上的军团圆点 |
| 行军 | 先选己方军团，再点目标省（或点「行军至此」） |
| 征召 | 侧栏「征召军团」（在己方领地） |
| 宣战 | 选敌方省份后「对领主宣战」 |
| 推进 | 顶栏 +1天 / 周 / 月 / 季 / 年 |
